Виртуальный плоттер (функциональный стиль Clojure)

Переписал процедурный код из plotter.ts в чисто функциональном стиле.

Что сделано:
- Полностью immutable состояние
- Чистые функции (каждая возвращает новый state)
- Threading-макрос ->>
- Вывод в консоль в соответствии с оригиналом
- Нет глобальных переменных 

Запуск: (load-file "plotter.clj") → (run)
